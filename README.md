# 🎯 VibeCheck – Frontend

VibeCheck is a **React-based frontend application** designed for students in Manipal to discover **food spots, hangout places, and campus events** based on different *vibes* like *Late Night*, *Budget*, *Chill*, etc.

This repository contains the **frontend-only implementation**.

---

## 🚀 Features

* 🔍 **Search** places by name
* 🎯 **Vibe-based filtering** (Late Night, Budget, Aesthetic, etc.)
* 🧱 Responsive **card-based layout**
* 🖼️ Local image support (no external APIs)
* 🎨 Clean UI with Bootstrap and custom CSS
* 🧩 Modular React component structure

---

## 🛠️ Tech Stack

* **React.js** (Functional Components + Hooks)
* **JavaScript (ES6+)**
* **Bootstrap 5** (utility classes)
* **CSS** (custom styling)
* **Static JSON** as mock data source

---

## ▶️ How to Run the Project Locally

### 1️⃣ Clone the repository

```bash
git clone https://github.com/richashanbhag/VibeCheck.git
```

### 2️⃣ Navigate into the project folder

```bash
cd vibecheck-frontend
```

### 3️⃣ Install dependencies

```bash
npm install
```

### 4️⃣ Start the development server

```bash
npm start
```

The app will run at:

```
http://localhost:3000
```
